# Number Guessing Game

## Description

A Python-based command-line Number Guessing Game where the user attempts to guess a randomly generated number.
The game offers multiple difficulty levels, providing an engaging and interactive experience while demonstrating core programming concepts.

## Features

* Multiple difficulty levels:

  * Easy (1–50 with 10 chances)
  * Medium (1–100 with 7 chances)
  * Hard (1–1000 with 5 chances)
* Random number generation for each game
* Real-time feedback:

  * "Too High" / "Too Low" hints
* Score calculation based on remaining attempts
* Replay option to play multiple rounds
* Input validation for user-friendly experience

## How to Run

1. Install Python (if not already installed)
2. Save the file as `game.py`
3. Open terminal or command prompt
4. Run the program:

```bash id="z84hh3"
python game.py
```

## Requirements

* Python 3.x
* Uses only standard Python libraries (`random` module)

## Learning Outcomes

* Understanding of loops and conditional statements
* Working with user input and validation
* Implementing game logic and scoring systems
* Using random number generation in Python
